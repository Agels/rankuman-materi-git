Intro GIT

Apa itu git?

  • sebuah control system untuk mencatat perubahan

Apa manfaat git?

  • mempermudah melakukan colaborasi antar developer
  • bisa menjadi backup source code

git commit -m "intro git"

fundamental git

git command via terminal

  • git init

    • untuk inisialisasi repo

  • git add

    • untuk memindah kan file ke dalam stage

  • git status

    • untuk melihat status repo

  • git log

    • untuk melihat history di dalam repo

  • git commit

    • untuk melakukan commit perubahan atau penambahan file

  • git clone

    • untuk mengcopy repository dari git

  • git remote

    • untuk melakukan koneksi ke repo yang di inginkan

  • git push

    • untuk mengirim commit ke repository cloud

  • git config

    • untuk melakukan configurasi email dan username

git commit -m "git fundamental"

Git branch

fungsi branch pada git

  • dapat digunakan untuk test fitur baru
  • untuk memenajemen kolaborasi antar developer

git command

  • git branch

    • untuk melihat semua brand yang ada pada local

  • git branch --all

    • untuk melihat semua branch global

  • git branch (nama_branch)

    • untuk menambah branch baru

  • git checkout (nama_branch)

    • untuk berpindah branch

  • git branch -d (nama_branch)

    • untuk menghapus branch

conflict pada git

  • ketika 2 branch atau lebih merubah file yang sama
  • ketika terjadi conflict antar branch tidak bisa melakukan commit
  • untuk commit conflict antar branch harus clear

    • dengan cara memilih salah satu branch yang akan di commit

git commit -m "git branch"

gui

fungsi gui

  • untuk mempermudah mengoperasikan git
  • mempercepat kinerja
  • solving conflict branch lebih mudah

git commit -m "gui tools"
