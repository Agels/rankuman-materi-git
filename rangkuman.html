fundamental git

git command via terminal

  • git init

    • untuk inisialisasi repo

  • git add

    • untuk memindah kan file ke dalam stage

  • git status

    • untuk melihat status repo

  • git log

    • untuk melihat history di dalam repo

  • git commit

    • untuk melakukan commit perubahan atau penambahan file

  • git clone

    • untuk mengcopy repository dari git

  • git remote

    • untuk melakukan koneksi ke repo yang di inginkan

  • git push

    • untuk mengirim commit ke repository cloud

  • git config

    • untuk melakukan configurasi email dan username
